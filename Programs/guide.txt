OpCodes

00XX : LD A RAM
08XX : LD B RAM
04XX : LD A IMM LPW
06XX : LD A IMM HIGH
0CXX : LD B IMM LOW
0EXX : LD B IMM HIGH

10XX : ST A
18XX : ST B
20XX : INC A
28XX : INC B

30XXX : BR

4000 : ADD
5000 : SUB
6000 : OR
7000 : AND
8000 : XOR
9000 : SHR
A000 : MOV
B000 : EXCH

C000 : CMP
DX00 : SET
EX00 : CLR
